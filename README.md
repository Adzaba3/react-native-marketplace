- React
- NativeWind
- Typescript
- ES Lint
- Babel


### Get started (npm)

1) Clone Repo
```
git clone https://github.com/Adzaba3/react-native-marketplace.git

cd react-native-marketplace
```

2) Install Dependencies
```
npm install
```

3) Run a script  
```
npx expo start
```
